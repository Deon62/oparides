# OPA Landing Page

Official landing page for OPA - On-Demand Driver & Car Rental Platform.

## Features

- Modern, minimal, and professional design
- Fully responsive layout
- Smooth scroll navigation
- Download buttons for Android and iOS
- Mobile-friendly interface

## Setup

1. Simply open `index.html` in a web browser, or
2. Deploy to Vercel or any static hosting service

## Deployment to Vercel

1. Push this folder to a GitHub repository
2. Connect the repository to Vercel
3. Set the build command (if needed): None (static site)
4. Set the output directory: `/` (root of the repository)
5. Deploy!

## Customization

### Update Android APK Link

In `index.html`, update the Android download button href:
```html
<a href="YOUR_APK_URL" class="download-btn android-btn" id="androidBtn">
```

### Update iOS Message

The iOS button currently shows a "Coming Soon" modal. To update the message, edit the modal content in `index.html`.
